# User Guide

## Getting Started

### 1. Upload Your Data
- Use the sidebar to upload CSV files
- Required: Delivery Self-Schedule CSV and Pickup Requests CSV
- Optional: Request Assistance CSV

### 2. Configure Settings
- Verify warehouse address in sidebar
- Adjust any custom settings as needed

### 3. Schedule Deliveries
- Go to "Schedule Deliveries" tab
- Select target delivery date
- Click "Auto-Select Best Matches"
- Review selected deliveries and nearby pickups

### 4. Generate Routes
- Switch to "Optimized Routes" tab
- Select date for route generation
- Click "Generate Optimized Route"
- Review route schedule with times

### 5. Export and Navigate
- Download route manifest for signatures
- Add route to Google Calendar
- Open optimized route in Google Maps

## Features Overview

### Smart Scheduling
- Automatically matches client preferences
- Groups deliveries by geographic area
- Finds nearby pickups to maximize efficiency

### Route Optimization
- Uses Google Maps for real-time traffic
- Implements EAGCA algorithm for area clustering
- Schedules within TFP operating hours

### Analytics
- Track overdue requests (8+ weeks)
- Monitor coverage gaps by zip code
- Generate comprehensive reports

### Data Management
- View and filter all CSV data
- Download filtered datasets
- Track scheduling performance

## Best Practices

### Data Preparation
- Ensure dates are in M/D/YYYY format
- Include complete addresses for pickups
- Keep phone numbers in consistent format

### Route Planning
- Schedule 3-4 deliveries per route maximum
- Allow 2-3 pickups per route
- Consider traffic patterns for timing

### Quality Control
- Review auto-selected deliveries
- Verify pickup addresses are accurate
- Check route makes geographic sense

## Troubleshooting

### No Deliveries Found
- Check if dates in CSV match selected date
- Verify CSV format is correct
- Try nearby dates if no matches

### Route Generation Fails
- Ensure addresses are complete
- Check internet connection for geocoding
- Verify warehouse address is set

### Google Maps Issues
- Ensure addresses include city and state
- Check for special characters in addresses
- Try refreshing the page
